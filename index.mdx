## Quick Start

<Steps>
  <Step title="Create an Account">
    Sign up at [JoggAI Console](https://dashboard.jogg.ai) to create your account
  </Step>
  <Step title="Get API Key">
    Navigate to API section in your dashboard to obtain your API key
    
    ```bash
    curl -X GET 'https://api.jogg.ai/open/v2/user/whoami' \
      -H 'x-api-key: YOUR_API_KEY'
    ```
  </Step>
  <Step title="Create Your First Video">
    Use the API to create a product video
    
    ```bash
    curl -X POST 'https://api.jogg.ai/open/v2/product' \
      -H 'x-api-key: YOUR_API_KEY' \
      -H 'Content-Type: application/json' \
      -d '{
        "name": "Your Product",
        "description": "Product description"
      }'
    ```
  </Step>
  <Step title="Check Video Status">
    Monitor your video generation progress
    
    <Check>
    Video generation typically takes 3-8 minutes
    </Check>
  </Step>
</Steps>

## Consumption

### Video Generation Credits

**a. Generated video consumption based on duration:**
- 1 API credit per 2 minutes of video
- Less than 2 minutes = 1 API credit

**b. Preview video consumption:**
- 0.5 API credit per 2 minutes
- Less than 2 minutes = 0.5 API credit

**c. AI Scripts API:**
- 0.2 API credit per call

**d. Image Generation:**
- Each call generates 4 images
- 0.05 credit per image
- Total: 0.2 credit per call

**e. Motion Generation:**
- Motion 1.0: 2 credits
- Motion 2.0: 2 credits
- Motion 2.0 Pro: 3 credits
