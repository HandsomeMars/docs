## How It Works

AI script generation is an **asynchronous process**:

<Steps>
  <Step title="Submit Request">
    Send product information or topic to the AI
  </Step>
  
  <Step title="AI Processing">
    AI analyzes content and generates multiple script options (10-30 seconds)
  </Step>
  
  <Step title="Retrieve Scripts">
    Get generated scripts via results endpoint
  </Step>
  
  <Step title="Select & Use">
    Choose your favorite script for video creation
  </Step>
</Steps>

## Quick Start

### Step 1: Generate Scripts

```bash
curl --request POST \
  --url 'https://api.jogg.ai/open/v2/ai_scripts' \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "product_name": "Amazing Smart Watch",
    "product_description": "A smartwatch with health tracking and 7-day battery",
    "product_features": [
      "Heart rate monitoring",
      "Sleep tracking",
      "Waterproof design"
    ],
    "script_type": "marketing",
    "language": "english",
    "duration": 30
  }'
```

**Response:**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "task_id": "task_abc123",
    "status": "processing"
  }
}
```

<Check>
  Save the `task_id` to retrieve your generated scripts!
</Check>

### Step 2: Get Generated Scripts

Wait 10-30 seconds, then retrieve your scripts:

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/ai_scripts/results/task_abc123' \
  --header 'x-api-key: YOUR_API_KEY'
```

**Response:**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "task_id": "task_abc123",
    "status": "completed",
    "scripts": [
      {
        "script_id": "script_1",
        "script": "Meet the Amazing Smart Watch - your ultimate health companion! Track your heart rate, monitor your sleep, and stay connected all day...",
        "duration": 28,
        "word_count": 85
      },
      {
        "script_id": "script_2",
        "script": "Revolutionize your fitness journey with the Amazing Smart Watch. Advanced sensors track every beat of your heart...",
        "duration": 30,
        "word_count": 90
      }
    ]
  }
}
```

### Step 3: Use Script in Video

Use the generated script to create a video:

```bash
curl --request POST \
  --url 'https://api.jogg.ai/open/v2/create_video_from_avatar' \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "script": "Meet the Amazing Smart Watch - your ultimate health companion!...",
    "avatar_id": 127,
    "voice_id": "en-US-ChristopherNeural",
    "aspect_ratio": 0
  }'
```

## Request Parameters

### Required

| Parameter | Type | Description |
|-----------|------|-------------|
| `product_name` | string | Name of your product or topic |
| `script_type` | string | Type: `marketing`, `educational`, `storytelling`, `social` |
| `language` | string | Language for the script (e.g., `english`, `spanish`) |
| `duration` | number | Target duration in seconds (15-180) |

### Optional

| Parameter | Type | Description |
|-----------|------|-------------|
| `product_description` | string | Detailed product description |
| `product_features` | array | List of key features |
| `product_url` | string | Product URL for AI to analyze |
| `tone` | string | Tone: `professional`, `casual`, `enthusiastic`, `calm` |
| `target_audience` | string | Target audience description |
| `call_to_action` | string | CTA to include at the end |

## Tips for Better Scripts

**Provide Rich Context**
- Include detailed product descriptions
- List specific features and benefits
- Mention your target audience
- Specify desired tone and style

**Optimize Duration**
- 15-30s: Quick social media content
- 30-60s: Standard marketing videos
- 60-120s: Detailed product demos
- 120-180s: In-depth tutorials

**Use Product URLs**
- AI can analyze web pages for more context
- Works best with product pages and landing pages
- Include comprehensive product information on the page

**Review & Refine**
- AI generates multiple variations - choose the best one
- Edit scripts to match your brand voice
- Test different tones and styles
- Combine elements from different versions

## Common Issues

<AccordionGroup>
  <Accordion title="Script Generation Timeout">
**Problem:** Task status stuck on "processing"

**Solutions:** Wait up to 60 seconds for complex requests. Check task status periodically. If it takes too long, simplify your input or try again with a shorter duration.

  </Accordion>
  
  <Accordion title="Script Too Long or Short">
**Problem:** Generated script doesn't match duration

**Solutions:** Adjust the `duration` parameter and regenerate. If needed, edit the script manually or try different script types for better results.

  </Accordion>
  
  <Accordion title="Script Quality Issues">
**Problem:** Script doesn't match expectations

**Solutions:** Provide more detailed product description and specify `tone` and `target_audience`. Try different `script_type` options. Generate multiple variations and combine the best parts.

  </Accordion>
  
  <Accordion title="Language Not Supported">
**Problem:** Desired language unavailable

**Solutions:** Check the supported languages list first. You can generate in English and translate externally, or contact support for language requests.

  </Accordion>
</AccordionGroup>
