## Introduction

Create personalized avatar videos using custom photo avatars. Transform any portrait photo into a talking avatar that can be used in your videos.

<Info>
  This feature requires you to first create a photo avatar. See [Create Photo Avatar](/api-reference/workflows/create-photo-avatar) for details.
</Info>

## Prerequisites

Before creating videos with photo avatars, you need:

1. ✅ A generated photo avatar (`avatar_id`)
2. ✅ A voice for the avatar (`voice_id`)
3. ✅ Your video script

## Quick Start

### Step 1: Get Your Photo Avatar ID

If you've already created a photo avatar, retrieve its ID:

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/avatars/photo_avatars' \
  --header 'x-api-key: YOUR_API_KEY'
```

**Response:**

```json
{
  "code": 0,
  "data": {
    "avatars": [
      {
        "avatar_id": "photo_123",
        "name": "My Custom Avatar",
        "status": "completed",
        "thumbnail_url": "https://res.jogg.ai/avatars/photo_123.jpg"
      }
    ]
  }
}
```

### Step 2: Create Video with Photo Avatar

```bash
curl --request POST \
  --url 'https://api.jogg.ai/open/v2/create_video_from_avatar' \
  --header 'x-api-key: YOUR_API_KEY' \
  --header 'Content-Type: application/json' \
  --data '{
    "script": "Hello! This is my custom photo avatar speaking.",
    "avatar_id": "photo_123",
    "avatar_type": "photo_avatar",
    "voice_id": "en-US-JennyNeural",
    "aspect_ratio": 0,
    "screen_style": 0
  }'
```

<Check>
  Note the `avatar_type: "photo_avatar"` parameter - this tells the API to use a photo avatar instead of a library avatar
</Check>

**Response:**

```json
{
  "code": 0,
  "msg": "success",
  "data": {
    "project_id": "vid_abc123"
  }
}
```

### Step 3: Retrieve the Video

```bash
curl --request GET \
  --url 'https://api.jogg.ai/open/v2/avatar_video/vid_abc123' \
  --header 'x-api-key: YOUR_API_KEY'
```

## Key Differences from Library Avatars

| Aspect | Library Avatars | Photo Avatars |
|--------|----------------|---------------|
| **avatar_type** | Not required (default) | `"photo_avatar"` required |
| **avatar_id** | Number (e.g., `127`) | String (e.g., `"photo_123"`) |
| **Quality** | Professional studio quality | Depends on source photo |
| **Customization** | Pre-made avatars | Your own photos |
| **Processing** | Instant | Requires photo avatar generation |

## Complete Workflow Example

<Steps>
  <Step title="Create Photo Avatar">
Upload a photo and generate the avatar:

```bash
curl -X POST 'https://api.jogg.ai/open/v2/photo_avatar/photo/generate' \
  -H 'x-api-key: YOUR_API_KEY' \
  -d '{
    "photo_url": "https://example.com/photo.jpg",
    "name": "John Doe Avatar"
  }'
```

Get `task_id`, then check status.
  </Step>
  
  <Step title="Get Avatar ID">
Once photo avatar is ready, get its ID:

```bash
curl -X GET 'https://api.jogg.ai/open/v2/avatars/photo_avatars' \
  -H 'x-api-key: YOUR_API_KEY'
```

Save the `avatar_id`.
  </Step>
  
  <Step title="Create Video">
Use the photo avatar in your video:

```bash
curl -X POST 'https://api.jogg.ai/open/v2/create_video_from_avatar' \
  -H 'x-api-key: YOUR_API_KEY' \
  -d '{
    "script": "Your script here",
    "avatar_id": "photo_123",
    "avatar_type": "photo_avatar",
    "voice_id": "en-US-JennyNeural",
    "aspect_ratio": 1
  }'
```
  </Step>
  
  <Step title="Get Result">
Retrieve the generated video via webhook or polling.
  </Step>
</Steps>

## Best Practices

**Photo Avatar Quality**
- Use high-quality source photos (at least 800x800)
- Ensure good lighting and clear facial features
- Avoid accessories that cover the face
- Use frontal photos for best results

**Voice Matching**
- Choose voices that match the avatar's appearance
- Test different voices to find the best fit
- Consider the gender and age of the avatar

**Video Settings**
- Use appropriate aspect ratios for your platform
- Add subtitles for better accessibility
- Test different screen styles for various use cases

<Tip>
  Photo avatars work best with scripts under 2 minutes. For longer content, consider splitting into multiple videos.
</Tip>

## Common Issues

<AccordionGroup>
  <Accordion title="Avatar Type Not Specified">
    **Error**: Video uses wrong avatar  
    **Solution**: Always include `"avatar_type": "photo_avatar"` when using photo avatars
  </Accordion>
  
  <Accordion title="Photo Avatar Not Ready">
    **Error**: Avatar not found  
    **Solution**: Check photo avatar status is "completed" before creating video
  </Accordion>
  
  <Accordion title="Quality Issues">
    **Problem**: Video quality not satisfactory  
    **Solution**: Use higher quality source photos and ensure good lighting
  </Accordion>
</AccordionGroup>
